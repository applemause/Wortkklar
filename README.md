# Wortklar

Минималистичный русско-немецкий и немецко-русский переводчик для изучения языка.

## Идея

Wortklar не только переводит, но и объясняет структуру языка:

- показывает артикль и род существительных;
- показывает множественное число;
- выводит три основные формы глагола;
- отмечает вспомогательный глагол и управление;
- объясняет различия употребления;
- даёт живые примеры с переводом;
- позволяет сохранять слова для повторения.

## Текущий статус

Рабочий переводчик с интерфейсом настроек, OpenAI API и структурированным
грамматическим разбором. API-ключ вводится пользователем в браузере и передаётся
серверному маршруту только во время запроса.

## Запуск

1. Установить Node.js 20 или новее.
2. Выполнить `npm install`.
3. Выполнить `npm run dev`.
4. Открыть `http://localhost:3000`.

## Развёртывание на Cloudflare

Проект использует серверный маршрут `/api/translate`, поэтому его нельзя
развёртывать как статический Next.js Export в Cloudflare Pages. Для полного
Next.js-приложения используется Cloudflare Workers и официальный OpenNext-адаптер.

Для Git-интеграции Cloudflare Workers укажите:

- production branch: `main`;
- root directory: `/`;
- build command: `npm run build:cloudflare`;
- deploy command: `npx opennextjs-cloudflare deploy`;
- output directory: не указывать.

Локальная проверка Worker-сборки:

```bash
npm ci
npm run preview
```

Ручное развёртывание из авторизованной среды:

```bash
npm ci
npm run deploy
```

Важно: не выбирайте пресет `Next.js (Static HTML Export)` и не задавайте `.next`
как папку публикации. `.next` — промежуточный результат Next.js, а готовый Worker
создаётся адаптером в `.open-next`.
